Hexo config配置文件说明

Site
设置 描述
title 您网站的标题
subtitle 您网站的字幕
description 您的网站说明
keywords 您网站的关键字。用逗号分隔多个关键字,。
author 你的名字
language 您网站的语言。使用2个字母的ISO-639-1代码或可选的其变体。默认值为en。
timezone 您网站的时区。Hexo默认使用计算机上的设置。您可以在此处找到可用时区列表。一些例子是America/New_York，Japan和UTC。

URL

设置 描述 默认
url 您网站的网址
root 您网站的根目录
permalink 文章的永久链接格式 :year/:month/:day/:title/
permalink_defaults 固定链接中每个细分的默认值
pretty_urls 将permalink变量重写为漂亮的URL
pretty_urls.trailing_index 尾随index.html，设置false为删除 true
pretty_urls.trailing_html 尾随.html，设置为false将其删除（不适用于尾随index.html） true

Directory
设置 描述 默认
source_dir 源文件夹。您的内容存储的位置 source
public_dir 公用文件夹。静态网站将在哪里生成 public
tag_dir 标签目录 tags
archive_dir 档案目录 archives
category_dir 分类目录 categories
code_dir 包含代码目录（的子目录source_dir） downloads/code
i18n_dir i18n目录 :lang
skip_render 将被复制到public原始路径而不进行渲染的路径。您可以使用glob表达式进行路径匹配。

Writing

设置 描述 默认
new_post_name 新帖子的文件名格式 :title.md
default_layout 默认布局 post
titlecase 将标题转换为标题大小写？ false
external_link 在新标签页中打开外部链接？
external_link.enable 在新标签页中打开外部链接？ true
external_link.field 适用于整体site或post仅 site
external_link.exclude 排除主机名。在适用时指定子域，包括www []
filename_case 将文件名转换为1小写；2大写 0
render_drafts 显示草稿？ false
post_asset_folder 启用资产文件夹？ false
relative_link 建立相对于根文件夹的链接？ false
future 显示未来的帖子？ true
highlight 代码块语法高亮设置，由Highlight.js提供支持
highlight.enable 启用语法突出显示 true
highlight.auto_detect 如果未指定语言，则启用自动检测 false
highlight.line_number 显示行号 启用此选项还将启用该wrap选项 true
highlight.tab_replace 用n个空格替换制表符；如果值为空，则不会替换标签 ''
highlight.wrap 将代码块包装在 `` true
highlight.hljs hljs-*为CSS类使用前缀 false

Home page setting

设置 描述 默认
index_generator 生成帖子存档，由hexo-generator-index提供支持
index_generator.path 博客索引页面的根路径 ''
index_generator.per_page 每页显示的帖子。 10
index_generator.order_by 发布订单。默认情况下按降序排列（从新到旧）。 -date
index_generator.pagination_dir URL格式，请参见下面的分页设置 page

Category&Tag

设置 描述 默认
default_category 默认类别 uncategorized
category_map 类别
tag_map 标签

Date/Time format

设置 描述 默认
date_format 日期格式 YYYY-MM-DD
time_format 时间格式 HH:mm:ss
use_date_for_updated post.updated如果前面没有提供更新日期，请使用发布日期。通常与Git工作流程一起使用 true

Pagination

设置 描述 默认
per_page 每页上显示的帖子数。0禁用分页 10
pagination_dir 网址格式 page

Extensions

设置 描述
theme 主题名称。false禁用主题
theme_config 主题配置。在此键下包括所有自定义主题设置，以覆盖主题默认设置。
deploy 部署设定
meta_generator 元生成器标签。false禁用标签的注入。

Inclide/Exclude Files or Folders

设置 描述
include 包括隐藏文件（包括名称/下划线开头的文件/文件夹，例外情况*）
exclude 排除文件/文件夹
ignore 忽略文件/文件夹
